This is an implementation of Git from scratch in Go, following the instructions of this article: https://codewords.recurse.com/issues/two/git-from-the-inside-out

It has every essential Git feature:

- init ✔
- add ✔
- rm ✔
- commit ✔
- fetch ✔
- pull ✔
- push ✔
- branch ✔
- checkout ✔
- diff ✔
- remote ✔
- merge ✔
- clone ✔
- .goitignore ✔
